🧩 How to Run Locally
# 1️⃣ Clone the project
git clone https://github.com/Pranathi-96/Ai-Course-Generator.git

# 2️⃣ Enter the folder
cd Ai-Course-Generator

# 3️⃣ Install dependencies
npm install

# 4️⃣ Run the dev server
npm run dev
